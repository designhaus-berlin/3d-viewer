    # 3D Viewer

    An iframe-based 3D viewer powered by Three.js. Embed it in any page and control it
    via a simple postMessage API — no Three.js knowledge required.

    ## Quick Start

    ```html
    <iframe
      id="viewer"
      src="https://designhaus-berlin.github.io/3d-viewer/viewer/index.html?model=YOUR_MODEL.glb"
      sandbox="allow-scripts allow-same-origin allow-pointer-lock allow-popups"
      allow="xr-spatial-tracking; fullscreen"
      referrerpolicy="no-referrer"
      style="width:100%; aspect-ratio:16/9; border:0;">
    </iframe>
    <script>
    const viewer = document.getElementById('viewer');
    function send(type, payload) {
      viewer.contentWindow.postMessage({ type, payload }, '*');
    }

    // Wait for viewer ready, then send commands:
    window.addEventListener('message', (e) => {
      if (e.data?.type === 'ready') {
        send('animateCamera', { position: {x:0, y:3, z:8}, target: {x:0, y:1, z:0} });
      }
    });
    </script>
    ```

    > **No model?** Leave out `?model=` and a demo scene appears automatically.

    > **[Live Demo](https://designhaus-berlin.github.io/3d-viewer/docs/index.html)**

    ---
